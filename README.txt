Harper Woods School District - Design Option B

Static prototype files:
- index.html: Homepage design option B
- school.html: Reusable Beacon Elementary school detail template
- assets/css/style.css: responsive CSS design system
- assets/js/main.js: vanilla JavaScript for navigation and anchor scrolling
- assets/images/: selected client assets from uploaded materials

Notes:
This is a design-first static prototype only. It does not use WordPress, PHP, Bootstrap, Tailwind, React, jQuery, paid plugins, or external UI kits.
